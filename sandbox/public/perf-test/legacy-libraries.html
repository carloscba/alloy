    Alloy - Web SDK

    Adobe Experience Platform Web SDK is a client-side JavaScript library that allows customers of the Adobe Experience Cloud to interact with the various services in the Experience Cloud through the Adobe Experience Platform Edge Network. In addition to the JavaScript library, there is a Launch extension to help with your Web SDK configurations.
